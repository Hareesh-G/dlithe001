Types of CSS

External CSS

with the external style sheet, we can change the look of entire website by changing just one file! each html page must include a referance to the external style sheet file inside the element, include an external style sheet can be written in text editor, and must be saved with .css extention. the external .css file should not contain any HTML page

Internal CSS

An internal style sheet may be used if one single HTML page has a unique style! The internal style is defined inside the style element, inside the head section

Inline CSS

AN inline style may be used to apply a unique style for a sigle element. To use inline styles, add the style attribute to the relevent element. The style attribute can contain any CSS property.
