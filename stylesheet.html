Introduction to CSS

CSS is the language we use to style a web page. CSS stands for Cascading Style Sheets CSS describes how HTML elements are to be displayed on screen, paper, or in other media CSS saves a lot of work It can control the layout of multiple web pages all at once. CSS is used to define styles for web pages, including the design, layout and variations in display for different devices and screen sizes.
